# 🏗️ Terraform Step-by-Step Learning

This directory contains our progressive Terraform learning journey, following official documentation.

## Directory Structure
```
terraform-step-by-step/
├── 01-hello-world/          # First Terraform configuration
├── 02-aws-connection/       # AWS provider setup  
├── 03-simple-s3/           # Deploy first resource (S3 bucket)
├── 04-lambda-basic/         # Lambda function
├── 05-api-gateway/          # API Gateway integration
├── 06-frontend-s3/          # Frontend hosting
├── 07-custom-domain/        # Route53 + SSL
├── 08-remote-state/         # S3 backend for state
└── 09-github-actions/       # Simple automation
```

## Learning Approach
1. **Start Simple**: Each step builds on the previous
2. **Follow Documentation**: Links to official docs in each folder
3. **Test Each Step**: Plan, apply, verify, destroy
4. **Understand Before Proceeding**: No black boxes
